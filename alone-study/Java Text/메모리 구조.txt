메모리구조

- stack 메모리 : 프로그램이 연산될때 메모리 사용량을 먼저 선점

- heap 메모리 : 프로그램이 연산 후에 실행될때 사용되는 메모리르 제공

--> 어떤 메모리를 쓰냐에 따라 다르다.

*) 대문자를 쓰면 참조 메모리 형태고 소문자를 쓰면 기본 데이터 타입이다.




	int age;	//	<-- 기본 데이터 타입
					// 메모리에 age에 저장할 데이터 공간을 할당한다.
		String name;// <-- 참조 데이터 타입
					// 메모리에 name에 저장할 객체 공간을 할당한다.
